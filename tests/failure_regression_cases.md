# 毕方失败样本回归用例

> 来源：网页版历史真实样例与用户测试反馈。用途：防止后续迭代把旧流程错误重新带回 skill。

## 回归表

| ID | 行业/场景 | 失败表现 | 根因归类 | 正确拦截 |
|---|---|---|---|---|
| FR-01 | 三农 / 平谷大桃 | 水果内容出现“根治/治疗/修复屏障/肤感清洁表达”等护肤风险套话 | 行业污染 / 风险错配 | `topic` 和 `review` 必须打回，风险边界必须按生鲜行业重写 |
| FR-02 | 三农 / 平谷大桃 | `用户为什么会看` 为空，但前台仍展示 `viralElement/score/爆款元素` | 旧流程污染 | 不得进入脚本或交付；必须补用户决策阶段和观看理由 |

| FR-04 | 三农 / 平谷大桃 | CTA 只有“扣1发链接/扣1发规格表”，没有筛选问题、规格确认或售后边界 | 承接错 | 降低硬成交感，改成规格/成熟度/发货时间/坏果处理等低压承接 |
| FR-05 | 三农 / 平谷大桃 | “实测糖度 XX°”或 `XX°` 占位符进入脚本、道具、字幕 | 证据错 / 虚构数字风险 | 未实测数字只能写“拍摄当天实测”，不能写进口播事实 |
| FR-06 | 洗护 / 冷制皂 | 目标用户写“敏感肌/过敏肌/激素脸/痘肌”，并承诺修复、祛痘、控油、减少闭口 | 高风险合规 | 用使用场景、肤感体验、清洁边界替代医学/功效标签 |
| FR-07 | 洗护 / 冷制皂 | CTA 写“肤质+年龄，我帮你配一套/定制方案” | 承接错 / 诊断承诺 | 只能引导使用场景、成分顾虑、试用边界，不能做肤质诊断或结果承诺 |
| FR-08 | 洗护 / 冷制皂 | 开头直接搬“非遗传承人”等身份背书，但没有过程证据 | 表达广告化 | 身份只能做辅助证据，必须先给用户问题、过程、可验证细节 |
| FR-09 | 本地餐饮 / 宝妈馄饨 | 出现“最安全/零添加/儿童专用/增强免疫力/促进发育”等表述 | 高风险合规 | 用配料表、现包过程、可核实食材来源替代绝对化和健康功效 |
| FR-10 | 通用 | 痛点只在口播里说，没有对应镜头、动作或现场证据 | 可拍性失败 / 证据错 | 脚本必须让关键判断有画面、动作、物料或现场过程支撑 |

## 必须回归的判断

- 有问题字段非空，不能给“通过”。
- 有占位符、待填数字、未验证结果，不能写成事实。
- 高风险行业先处理表达边界，再写承接动作。
- CTA 必须服务用户下一步判断，不是默认“扣1/私信发链接”。
- 身份背书不能替代证据；行业风险边界不能跨行业套用。
