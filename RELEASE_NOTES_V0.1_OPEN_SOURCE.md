# 毕方 V0.1 开源候选版发布说明

版本：`0.1.0`

## 定位

毕方 V0.1 开源版是一组商业短视频编导 skills。它的重点不是自动生成大量脚本，而是帮助用户先判断：

1. 目标用户真正想看什么。
2. 今天先拍哪 3 条。
3. 第一条内容改稿时不能丢什么。

## 适合谁

- 短视频编导、代运营、顾问。
- 本地商家、建材家装、B2B、课程咨询、手艺定制、电商和本地生活从业者。
- 有产品、服务、项目、课程或内容资产，但不知道先拍什么的人。

## 开源版包含

- 通用短视频商业内容判断 skill。
- 通用选题、脚本、审核、改稿、复盘模块。
- 开源验收和行业烟测样例。
- README、LICENSE 和基础发布说明。

## 开源版不包含

- 私有达人/竞品研究资料。
- 私有客户案例和发布数据。
- 内部商业化、报价、试跑材料。
- 尚未清理的复杂编排器。

## 推荐测试输入

```text
我是自己做内容/帮客户做内容。
我做什么行业，主要推广什么。
目标用户是谁。
我想通过短视频得到什么结果。
我目前有什么真实证据或素材。
```

## 请重点反馈

- 它有没有让你更清楚该拍什么。
- 它有没有说清你的用户真正想看什么。
- 给的 3 条先拍内容，你是否愿意真的拍第一条。
- 哪句话不准、太虚、看不懂或像普通文案工具。

## 风险边界

- 高风险行业只输出方向判断、流程透明和适用边界，不承诺疗效、收益或确定结果。
- 缺少案例、数据、资质和素材时，毕方应该标注待确认，不应该编造。
- 使用者需要自行确认平台规则、行业法规和素材真实性。
