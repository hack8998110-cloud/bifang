## 8. 不通过发布的条件

出现以下任一情况，不应发布 V0.2 正式版：

- 开源包混入私有达人资料、客户数据、内测压测明细。
- README 仍把毕方描述成泛文案工具或拍剪建议工具。
- starter 默认直接生成完整脚本，而不是先判断该拍什么。
- 高风险行业出现结果承诺。
- report 把待补证据内容包装成可交付。
- review 对客户改坏的脚本给通过。

毕方 V0.2-rc 通过能力验收。
